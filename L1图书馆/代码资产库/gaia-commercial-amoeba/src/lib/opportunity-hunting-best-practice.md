## 二、信号捕捉→验证→变现流水线

```
信号源 ──→ 每日扫描 ──→ AI初筛 ──→ OSCAR调研 ──→ 报告产出 ──→ 上架售卖
  │          │           │            │             │            │
  │    各阿米巴单元    DeepSeek分析  5阶段深挖   定价¥300-2000  gaia平台
  │    10分钟/天      30分钟/周     60分钟/次    1天/份       24h在线
```

### Step 1 — 每日信号扫描（10分钟/天）

各阿米巴单元每天早上用 AI 扫描2-3个信号源：

```python
# 示例：情报部扫描知乎热门话题
prompt = """
扫描2026年6月知乎上「商业分析」「市场趋势」「行业机会」等话题下
热度最高、讨论最热烈的10个问题。
输出JSON：问题标题、关注人数、是否蕴含商业需求、需求描述
"""
```

### Step 2 — AI初筛（30分钟/周）

周末AI汇总全部信号，三维评分：

| 维度 | 评分标准 | 权重 |
|:-----|:---------|:----:|
| 🔥 **热度** | 讨论人数、搜索量、增长率 | 30% |
| 💰 **付费意愿** | 用户是否愿意为解决方案付费 | 40% |
| 🏆 **竞争优势** | 竞品是否已满足、我们的差异化 | 30% |

**阈值**：综合≥7/10 → 进入OSCAR调研管道

### Step 3 — OSCAR深调（60分钟/次）

用升级后的 OSCAR 五阶段调研：

```
O 目标: 这个需求来自谁？TA愿意付多少钱？
S 范围: 这个需求有多大市场？增速？
C 清单: 谁已经在满足？他们怎么收费？
A 情报: 竞品月活？定价？用户评价？
R 判断: 我们做不做得？怎么做？多久回本？
```

### Step 4 — 快速产出（1天/份）

调研确认有机会 → 12小时内产出付费报告：

- **定价公式**: `价格 = 用户痛点强度 × 信息稀缺度 × 竞品定价`
- **基准价**: 简单报告¥300 / 深度研报¥800 / 定制咨询¥2000

### Step 5 — 上架售卖（24h在线）

gaia-commercial 平台自动上架，每笔销售自动记录到阿米巴营收。

## 六、铁律

| # | 铁律 | 说明 |
|:-:|:-----|:------|
| 1 | **每日扫描不可断** | 信号断了=收入断了，cronjob保底 |
| 2 | **证据等级不撒谎** | 每条情报标确凿/可信/推测，误导=自毁 |
| 3 | **先验证再生产** | 至少3个独立信号确认需求再产出报告 |
| 4 | **40%转化率底线** | 报告产出后真实售卖转化率<40%要复盘 |
| 5 | **复盘必须归档** | 失败的需求调研也要归档，教训是资产 |

---

**总结**: 不是坐在那里想「用户需要什么」，而是去知乎看用户在问什么、去GitHub看在吵什么、去招聘网站看在招什么。信号就在那里，阿米巴要做的是→发现→验证→产出→赚钱。
