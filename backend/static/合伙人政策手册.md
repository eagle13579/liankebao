> 副品牌：企盟 | 定位：企业家供需匹配平台
> 目标：100万企业会员 | 种子期：先跑通10个核心推广员的交易闭环

## 一、三级合伙人体系总览

| 等级 | 称号 | 门槛 | 分润比例 | 核心权益 | 考核周期 |
|------|------|------|----------|----------|----------|
| L1 | 资源经纪人 | 0门槛，注册即享 | 50%直接分润 | 自购省钱、分享赚钱、专属推广码 | 无硬性要求 |
| L2 | 企业合伙人 | 月≥5单 或 团队≥10人 | 20%~30%团队管理奖 | L1所有权益 + 团队管理奖 + 优先选品权 | 月度 |
| L3 | 城市合伙人 | 团队≥50人 | 8%~15%区域分成 | L2所有权益 + 城市经营授权 + 总部战略会席位 | 季度/年度 |

### 1.1 L1 资源经纪人（入门层）

**定位**：链客宝AI的"毛细血管"，每个人身边的企业家人脉都是资产。

**权益明细**：
- ✅ 自购返利：自己在平台消费，享受**零售价50%返还**
- ✅ 分享佣金：推荐他人成交，获得**成交额50%的佣金**
- ✅ 专属推广码：一人一码，永久绑定上下级关系
- ✅ 免费培训：每周三晚"推广大师课"（线上）
- ✅ 爆品优先体验权：每月2款新品免费试用名额
- ✅ 推广素材库：朋友圈文案、海报、短视频模板，每周更新

**分润示例**（以¥1,980爆品套餐为例）：
- 推广成交1单：佣金 = 1,980 × 50% = **¥990**
- 推广成交10单：佣金 = 9,900元
- 转介绍下级：下级成交，上级拿**团队管理奖**（仅L2及以上）

**考核标准**：无硬性考核，但连续3个月无活跃记录将失去爆品体验权。

### 1.2 L2 企业合伙人（骨干层）

**定位**：链客宝AI的"腰部力量"，带团队作战，放大收益。

**晋升门槛**（满足其一即可）：
- A. 个人月推广成交 ≥ 5单
- B. 直属团队总人数 ≥ 10人
- C. 个人+团队月总GMV ≥ ¥30,000

**权益明细**（包含L1全部权益，额外增加）：
- ✅ **团队管理奖**：直属L1团队成员每成交一单，L2可获得**成交额20%~30%**的管理提成
  - 团队月GMV < 5万 → 20%
  - 团队月GMV 5万~20万 → 25%
  - 团队月GMV > 20万 → 30%
- ✅ **优先选品权**：每月新品上架前，L2可提前48小时选品锁货
- ✅ **企业合伙人证书**：链客宝AI官方授权证书（含编号，可查）
- ✅ **总部线下沙龙**：每月1次，与大健康/企服头部企业创始人面对面
- ✅ **专属客服通道**：1对1运营顾问，响应时间<2小时

**考核标准（月度）**：

| 考核项 | 合格线 | 激励挂钩 |
|--------|--------|---------|
| 个人月成交 | ≥3单 | 低于3单，下月管理奖降至15% |
| 团队新增人数 | ≥3人/月 | 低于3人，次月优先选品权暂停 |
| 团队月总GMV | ≥¥15,000 | 低于¥15,000，管理奖按20%最低档 |

> ⚠️ 连续2个月不达标，降级为L1；连续3个月达标，晋升L3候选池。

### 1.3 L3 城市合伙人（战略层）

**定位**：链客宝AI的"一方诸侯"，掌控城市运营权，享受区域增长红利。

**晋升门槛**（须全部满足）：
- ✅ 团队总人数 ≥ 50人
- ✅ 其中L2（直属下级）≥ 5人
- ✅ 团队月度总GMV ≥ ¥100,000（连续3个月）
- ✅ 通过总部面试/答辩

**权益明细**（包含L1+L2全部权益，额外增加）：
- ✅ **区域分成**：所辖城市/区域所有订单的**8%~15%**流水分成
  - 区域月GMV < 50万 → 8%
  - 区域月GMV 50万~200万 → 10%
  - 区域月GMV 200万~500万 → 12%
  - 区域月GMV > 500万 → 15%
- ✅ **城市经营授权**：可使用"链客宝AI城市合伙人"品牌开展招商会、线下活动
- ✅ **总部战略会席位**：每季度参加链客宝AI战略闭门会，参与产品定价与选品决策
- ✅ **股权/期权激励池**：年度考核前10%的城市合伙人，获得链客宝AI期权资格（详情签署NDA后披露）
- ✅ **区域招商权**：可在本城市发展L2/L1团队，招商佣金归L3所有
- ✅ **线下活动补贴**：每月¥2,000~¥10,000城市运营基金（实报实销）
- ✅ **总部高管拜访**：每季度1次，CEO/COO亲自到访城市协助打单

**考核标准（季度/年度）**：

| 考核项 | 合格线 | 未达标后果 |
|--------|--------|-----------|
| 团队月均GMV | ≥¥100,000 | 连续2季度不达标，降为L2 |
| 团队人数增长 | 月增≥5人 | 连续3个月不增长，调整区域范围 |
| L2骨干培养 | 每季度新增≥2名L2 | 缺少L2梯队，区域分成降档1级 |
| 城市活动 | 每季度≥2场线下沙龙/招商会 | 活动场次不足，取消城市基金补贴 |
| 客户满意度 | 投诉率<2% | 投诉率过高，进入观察期 |

## 三、分润体系详解

### 3.1 L1分润模型（直推）

```
L1推广员 → 客户成交
         → 佣金 = 成交额 × 50%
         → 即时到账推广员钱包（T+1可提现）
```

### 3.2 L2分润模型（直推 + 团队管理）

```
L2推广员 → 自己成交 → 佣金 = 成交额 × 50%
         → 直属L1成交 → 团队管理奖 = 成交额 × 20%~30%（取决于团队月GMV档位）
         → 收益 = 个人佣金 + 团队管理奖
```

### 3.3 L3分润模型（直推 + 团队管理 + 区域分成）

```
L3合伙人 → 自己成交 → 佣金 = 成交额 × 50%
         → 直属L1/L2成交 → 团队管理奖（同上）
         → 区域所有订单 → 区域分成 = 区域总GMV × 8%~15%
         → 收益 = 个人佣金 + 团队管理奖 + 区域分成
```

### 3.4 分润规则注意事项

| 规则 | 说明 |
|------|------|
| 分润上限 | 同一订单，三级分润总和 ≤ 商品毛利的80%，保障平台健康运营 |
| 平级不抽 | L2下级晋升为L2后，原上级不再抽取该L2团队的管理奖，该L2团队归L3统一管理 |
| 越级保护 | 上级L3不可直接"抢"下级L2的直推客户，客户归属以首次绑定为准 |
| 分润结算周期 | 个人佣金T+1到账钱包；团队管理奖T+7；区域分成月结 |
| 退款处理 | 客户退款，对应订单的分润全额追回（从后续佣金中扣除） |

## 四、种子用户专项政策（第一批推广员专属）

**目标**：首批招募10位种子推广员，跑通交易闭环

### 4.1 种子用户特权（限前10名）

- 🎁 **终身分润加码**：种子用户永久享受L2级别的团队管理奖比例（即使仅达L1标准）
- 🎁 **创始编号**：独立编号 #0001~#0010，链客宝AI历史永存
- 🎁 **零成本创业包**：平台提供¥500推广体验金（可用于购买样品）
- 🎁 **创始人直通车**：可直接与链客宝AI创始人1对1微信沟通，每周反馈直达产品
- 🎁 **闭门私董会**：每月1次种子用户私董会，对接头部企业资源
- 🎁 **期权优先权**：第一批种子用户享有期权池优先认购权（估值最优惠期）

### 4.2 种子期玩法（前90天）

| 阶段 | 时间 | 目标 | 支持策略 |
|------|------|------|---------|
| 冷启动 | 第1~15天 | 每人完成≥3单成交 | 总部提供"种子大礼包"（含3份样品），协助邀约 |
| 放大期 | 第15~45天 | 每人发展≥5名下家 | 总部每周1场线上招商会，邀请种子用户做分享嘉宾 |
| 体系化 | 第45~90天 | 形成可复制的推广大SOP | 总部协助梳理流程，形成标准推广手册 |

### 4.3 种子用户退出机制

- 前90天内如连续30天无任何成交记录，种子特权自动取消，转为普通L1
- 如有违规行为（虚假交易、恶意刷单、传播不实信息等），立即取消资格并追回已发分润
