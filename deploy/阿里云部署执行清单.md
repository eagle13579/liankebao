### GitHub Secrets 配置说明

在 GitHub 仓库 Settings > Secrets and variables > Actions 中添加：

| Secret 名称 | 值 | 说明 |
|-------------|-----|------|
| `HOST` | `47.100.160.250` | 服务器 IP 地址 |
| `USERNAME` | `deploy` (或你的SSH用户名) | SSH 登录用户名 |
| `SSH_KEY` | SSH 私钥内容 | 建议使用专用部署密钥 |
| `DEPLOY_PATH` | `/opt/liankebao` | 部署目标路径 |

> 推荐创建一个专用的部署用户，限制其权限：
> ```bash
> sudo useradd -m -s /bin/bash deploy
> sudo mkdir -p /home/deploy/.ssh
> sudo chown -R deploy:deploy /home/deploy/.ssh
> sudo usermod -aG www-data deploy
> ```

## 紧急回滚方案

如果部署后出现问题：

```bash
# 回滚后端版本
cd /opt/liankebao
git log --oneline -5
git checkout <前一个稳定commit>
sudo systemctl restart chainke

# 回滚前端版本（保留之前的 dist 备份）
mv /opt/liankebao/dist /opt/liankebao/dist_new
cp -r /opt/liankebao/dist_bak /opt/liankebao/dist

# 回滚 nginx 配置
cp /etc/nginx/nginx.conf.bak /etc/nginx/nginx.conf
sudo nginx -s reload
```
