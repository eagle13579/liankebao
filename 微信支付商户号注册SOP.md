# 微信支付商户号注册 SOP — 链客宝专用

> 预计耗时：30-45分钟（准备材料）+ 15分钟（在线填写）  
> 审核周期：1-3个工作日  

## 前置准备：需要你手边有的材料

| 材料 | 说明 | 注意事项 |
|:-----|:-----|:---------|
| 营业执照 | 彩色照片或扫描件 | 清晰、四角完整、无反光 |
| 法人身份证 | 正反面彩色照片 | 在有效期内 |
| 对公银行账户 | 开户行、账号 | 个体户可用法人个人银行卡 |
| 手机号 | 法人实名手机号 | 用于接收验证码 |
| 邮箱 | 未注册过微信支付的邮箱 | 用于接收审核通知 |
| 小程序 AppID | wxb4f6d89904200fd2 | 已注册，直接拿来用 |

---

## Step 1：进入注册页面

1. 浏览器打开 https://pay.weixin.qq.com
2. 点击右上角「注册微信支付商户号」
3. 选择「我是普通商户」
4. 选择「微信小程序」→ 输入 AppID：wxb4f6d89904200fd2
5. 点击「下一步」

---

## Step 2：填写主体信息

### 2.1 主体类型
- 企业/个体户 → 选「企业」或「个体工商户」
- 如实选择，选错会导致审核不通过

### 2.2 营业执照信息
| 字段 | 填写说明 |
|:-----|:---------|
| 营业执照注册号/统一社会信用代码 | 执照上的18位代码 |
| 营业期限 | 执照上的有效期 |
| 营业执照照片 | 上传彩色扫描件/照片 |

### 2.3 法人信息
| 字段 | 填写说明 |
|:-----|:---------|
| 法人姓名 | 与身份证一致 |
| 法人身份证号 | 18位 |
| 身份证有效期 | 按身份证填写 |
| 身份证正反面 | 分别上传，清晰无反光 |

### 2.4 经营者/法人联系方式
- 手机号：法人实名手机号
- 邮箱：未在微信支付注册过的邮箱

---

## Step 3：填写经营信息

### 3.1 商户简称
链客宝

### 3.2 经营类目
推荐选「综合电商/电子商务平台」（费率0.6%，最适合链客宝）。

### 3.3 经营场景
- 选择「小程序」
- 填写 AppID：wxb4f6d89904200fd2

### 3.4 结算规则
- 结算周期：T+1
- 结算银行卡：对公账户（企业）/ 法人个人卡（个体户）

---

## Step 4：上传资质注意事项

| 容易出错的地方 | 正确做法 |
|:--------------|:---------|
| 照片模糊 | 用扫描件或用手机在光线充足时拍 |
| 执照边框裁切 | 四角完整，包含二维码区域 |
| 身份证反面没有国徽面 | 反面也要拍清楚 |
| 法人手机号非实名 | 必须运营商实名认证的手机号 |
| 联系人不是法人 | 首次申请建议法人自己操作 |

---

## Step 5：绑定小程序（审核通过后）

1. 登录 pay.weixin.qq.com → 产品中心 → AppID账号管理
2. 点击「关联AppID」，输入 wxb4f6d89904200fd2
3. 小程序管理员微信确认 → 点同意
4. 登录 mp.weixin.qq.com → 功能 → 微信支付 → 确认关联

---

## Step 6：获取配置参数

| 配置项 | 获取位置 | 用途 |
|:-------|:---------|:-----|
| WXPAY_MCHID | 商户平台首页「商户号」 | 商户身份标识 |
| WXPAY_API_V3_KEY | 账户中心→API安全→设置APIv3密钥 | 支付签名 |
| WXPAY_CERT_SERIAL_NO | 账户中心→API安全→证书管理 | 证书序列号 |
| WXPAY_CERT_PATH | 下载apiclient_key.pem | 请求签名 |
| WXPAY_PLATFORM_CERT_PATH | 下载平台证书 | 回调验签 |
| WXPAY_NOTIFY_URL | https://www.go-aiport.com/lkapi/wechat/pay/notify | 支付回调地址 |

---

## 常见驳回原因

- 营业执照信息不一致 → 检查统一信用代码
- 法人身份信息不一致 → 姓名+身份证号与公安系统一致
- 经营类目不匹配 → 改为「综合电商」
- 法人手机号非实名 → 去运营商确认实名状态

---

## 注册完成后操作

1. 把 WXPAY_MCHID / WXPAY_API_V3_KEY / WXPAY_CERT_SERIAL_NO 给白泽
2. apiclient_key.pem 放到 D:\链客宝\backend\certs\ 目录
3. 白泽配置完 .env 后，小程序即可提审发布
