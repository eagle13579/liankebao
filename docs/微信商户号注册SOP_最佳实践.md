# 微信商户号注册 SOP · 最佳实践（2026版）

> 目标：为链客宝小程序开通微信支付，完成商户号注册+认证+API配置
> 预估耗时：**1天操作 + 3-7天审核**
> 注意：以下流程**必须用公司主体**注册，个体户/个人主体在小程序支付场景下限制极多

## 一、前置准备（做完再开始注册）

### 1.1 必查清单

| 项目 | 要求 | 检查 |
|:-----|:-----|:----:|
| 公司营业执照 | 在有效期内，经营范围含"互联网信息服务"或相关 | □ |
| 法人身份证 | 正反面清晰照片 | □ |
| 对公银行账户 | 开户行/账号/开户许可证 | □ |
| 小程序AppID | 已认证的微信小程序（**必須已认证**，年费300元） | □ |
| 公司邮箱 | 未注册过微信商户平台的邮箱 | □ |
| 域名 | 已备案的HTTPS域名（liankebao.top ✅） | □ |
| 手机号 | 法人或经办人的实名手机号 | □ |

### 1.2 重要：小程序认证 vs 商户号认证

```
小程序认证（300元/年）≠ 商户号认证
   ├─ 小程序认证 = 微信开放平台认证，获得wx.xxxx AppID
   └─ 商户号认证 = 微信支付单独审核，审核费0元
两者独立！但小程序必须先认证才能开通微信支付
```

## 二、注册流程（分步截图指南）

### Step 1：进入微信商户平台

```
URL: https://pay.weixin.qq.com/
点击右上角「注册」→ 选择「注册微信支付商户号」
```

### Step 2：选择接入方式

```
选「微信小程序JSAPI支付」
   ├─ 需要：已认证的小程序AppID
   └─ 优势：用户在小程序内直接调起支付，无需跳出
```

### Step 3：填写主体信息

| 字段 | 填写规则 | 常见错误 |
|:-----|:---------|:---------|
| 主体类型 | 选「企业」 | 别选「个体工商户」（限额低） |
| 营业执照号 | 统一社会信用代码 | 字母大写，不要空格 |
| 法人姓名 | 与身份证完全一致 | 生僻字注意 |
| 对公账户 | 开户行全称+账号 | 支行名称写全，不要缩写 |
| 经营类目 | 选「信息技术服务-软件服务」 | 选错类目会导致费率不同 |

**费率提示**：
- 标准：0.6%（经营类目不同有差异）
- 优惠：部分行业/区域有0.38%的优惠费率
- 链客宝建议选：信息技术服务 → 0.6%

### Step 4：填写经营信息

```
商户简称：链客宝           （展示在用户支付页面的名称）
客服电话：你的手机号        （审核和用户联系用）
经营地址：营业执照地址      （与营业执照一致）
商户名称：链客宝企盟平台    （用于商户平台显示）
```

### Step 5：超级管理员绑定

```
超级管理员 = 能登录商户平台的人（建议填法定代表人或运营负责人）
  ├─ 姓名
  ├─ 手机号（实名认证的手机号）
  └─ 邮箱
  → 审核通过后，管理员的微信会收到通知
  → 扫码即可登录商户平台
```

### Step 6：提交审核

```
提交后：
  1. 微信支付团队会在3-7个工作日内审核
  2. 审核期间会有电话回访（注意接听0755开头的电话）
  3. 回访问题示例：
     - 「贵公司主营业务是什么？」→ 企业家供需对接平台
     - 「为什么开通微信支付？」→ 平台会员费/服务费收款
     - 「用户付款后会得到什么？」→ 会员服务/对接服务
  4. 审核通过后：微信会通过公众号/邮件通知

⏰ 加速技巧：
  - 工作日早上提交，当天可能进入审核队列
  - 确保法人/经办人的手机号能接听
  - 回访时不要说「充值和提现」「虚拟货币」等敏感词
```

## 三、注册完成后配置（重要！）

### 3.1 登录商户平台

```
https://pay.weixin.qq.com/
管理员微信扫码登录
```

### 3.2 获取API密钥

```
商户平台 → 账户中心 → API安全
  ├─ 设置APIv2密钥（32位，字母+数字）
  │   注意：这个密钥用于V2接口签名，非常重要！
  ├─ 设置APIv3密钥（32位）
  │   用于回调通知解密（可选但推荐）
  └─ 下载商户证书（p12/pem格式）
      证书路径：部署到服务器 /opt/liankebao/certs/
```

### 3.3 配置支付回调域名

```
商户平台 → 产品中心 → 开发配置
  └─ JSAPI支付授权目录
      https://liankebao.top/api/payment/
  └─ 支付回调域名
      https://liankebao.top
```

### 3.4 关联小程序AppID

```
商户平台 → 产品中心 → AppID账号管理
  └─ 关联已认证的小程序AppID
  └─ 确认关联（需要小程序管理员扫码确认）
```

## 五、常见问题

| 问题 | 原因 | 解决 |
|:-----|:-----|:-----|
| "营业执照已注册" | 该公司已完成注册 | 找回账号或联系客服解绑 |
| "经营类目不匹配" | 选的类目与执照经营范围不一致 | 按执照经营范围选择 |
| 审核被拒：经营范围不符 | 执照经营范围没写"互联网"相关 | 找代办或变更经营范围 |
| 审核被拒：资料模糊 | 照片不清楚/反光 | 重拍，确保四角可见 |
| 审核超7天无结果 | 排队或材料有问题 | 联系微信支付客服：95017 |
| 回访电话没接到 | 直接挂断会拒审 | 保持手机畅通，回拨95017解释 |
