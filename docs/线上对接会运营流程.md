# 线上闭门对接会 — 运营流程文档

## 一、概述

线上闭门对接会（Online Closed-Door Matching Event）是链客宝AI钻石会员的核心权益之一。
活动通过腾讯会议/飞书在线进行，为企业家提供精准、高效的资源对接体验。

**适用人群：** 钻石会员（diamond）及董事会会员（board）
**对接形式：** 线上（腾讯会议/飞书），每场限20-50人
**定位：** 会员专享、闭门、精准匹配

---

## 二、会员权益说明

### 钻石会员（¥599/年）
- 每月优先参与线上闭门对接会
- 会前获取参会企业名单
- 一对一精准匹配推荐
- 会后回放/资料共享

### 董事会会员（¥2999/年）
- 含所有钻石会员权益
- 可申请作为对接会分享嘉宾
- 专属商务顾问1对1服务

## 三、运营流程

### 3.1 活动创建（管理员后台）

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 确定主题 | 如"大健康产业资源对接专场" |
| 2 | 设定时间 | 建议提前7-14天发布，确保会员有充足时间报名 |
| 3 | 设定人数上限 | 建议20-50人，保证交流质量 |
| 4 | 准备会议链接 | 腾讯会议/飞书会议链接 |
| 5 | 创建活动 | POST /api/v1/events/online（管理员权限） |
| 6 | 发布活动 | status=published（默认） |

**创建活动参数：**
- title: 活动标题
- description: 活动描述（行业、参会企业介绍等）
- cover_image: 活动封面图（建议1920×720px）
- event_date: 活动开始时间（ISO 格式）
- end_date: 活动结束时间（ISO 格式）
- location: 会议链接
- max_participants: 最大参与人数
- tags: 标签，如"大健康,科技,融资"

### 3.2 用户报名流程

```
用户 → 会员中心 → 线上对接会 → 查看活动列表 → 选择活动 → 报名
                                                          │
                                                          ├── 钻石会员 → 报名成功
                                                          └── 非钻石会员 → 提示升级
```

**报名校验逻辑：**
1. 用户会员等级必须是 diamond 或 board
2. 会员未过期
3. 活动状态为 published
4. 活动未满员
5. 用户未重复报名

### 3.3 活动进行中

| 阶段 | 动作 | 负责人 |
|------|------|--------|
| 会前3天 | 发送参会提醒（短信/站内信） | 系统自动 |
| 会前1天 | 发送会议链接+参会须知 | 系统自动 |
| 会前30分钟 | 开放会议室 | 运营 |
| 活动进行 | 主持+控场+记录 | 运营 |
| 会后24h | 发送反馈问卷 | 系统自动 |

### 3.4 会后反馈收集

用户可在活动详情页提交反馈：
- 评分（1-5星）
- 评论文本（可选）

**反馈数据用途：**
- 优化活动质量
- 评估主持人/嘉宾表现
- 生成活动报告

## 四、API 接口文档

### POST /api/v1/events/online — 创建线上对接会
- 权限：管理员
- Body: CreateEventRequest

### GET /api/v1/events/online — 获取对接会列表
- 权限：无需登录
- Query: page, page_size, status

### GET /api/v1/events/online/{id} — 获取活动详情
- 权限：需登录（含是否已报名标识）

### POST /api/v1/events/online/{id}/register — 报名
- 权限：钻石会员/董事会会员
- Body: RegisterRequest

### POST /api/v1/events/online/{id}/feedback — 提交反馈
- 权限：已报名的用户
- Body: FeedbackRequest

## 五、数据库模型

### online_matching_events（线上对接会活动）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | Integer | 主键 |
| title | String(200) | 活动标题 |
| description | Text | 活动描述 |
| cover_image | String(500) | 封面图URL |
| event_date | DateTime | 活动日期 |
| end_date | DateTime | 结束日期 |
| location | String(200) | 会议链接 |
| max_participants | Integer | 最大人数 |
| current_participants | Integer | 当前报名人数 |
| price | Float | 价格（0=免费） |
| status | String(20) | draft/published/ongoing/completed/cancelled |
| tags | String(500) | 标签 |

### online_matching_registrations（报名记录）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | Integer | 主键 |
| event_id | Integer | 活动ID |
| user_id | Integer | 用户ID |
| status | String(20) | pending/confirmed/cancelled/attended |
| company | String(200) | 报名公司 |
| position | String(100) | 职位 |
| phone | String(20) | 电话 |
| notes | Text | 需求说明 |

### online_matching_feedback（反馈记录）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | Integer | 主键 |
| event_id | Integer | 活动ID |
| user_id | Integer | 用户ID |
| rating | Integer | 评分(1-5) |
| comment | Text | 反馈内容 |
| created_at | DateTime | 创建时间 |
