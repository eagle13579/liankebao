🔐 认证 Auth

用户注册、登录、令牌刷新和微信小程序登录。认证接口无需 Bearer Token。

POST /api/auth/login 用户登录
Body: { "username": "string", "password": "string" }
Response: access_token, refresh_token, user 信息
{
  "code": 200,
  "message": "登录成功",
  "data": {
    "access_token": "eyJ...",
    "refresh_token": "eyJ...",
    "token_type": "bearer",
    "user": { "id": 1, "username": "...", "role": "..." }
  }
}
POST /api/auth/register 用户注册
Body: { "username": "string", "password": "string", "name": "string", "phone?": "string", "company?": "string", "position?": "string", "role?": "string", "pain_point?": "string" }
Response: 注册成功，自动签发 token
GET /api/auth/me 获取当前用户 Bearer
Headers: Authorization: Bearer {{token}}
Response: 当前登录用户详细信息
POST /api/auth/refresh 刷新令牌
Body: { "refresh_token": "string" }
Response: 新的 access_token + refresh_token（轮换机制）
POST /api/auth/wechat 微信小程序登录
Body: { "code": "string" } 微信临时 code
Response: 登录成功，签发 token
POST /api/auth/logout 退出登录 Bearer
功能: 将当前 token 加入黑名单
POST /api/auth/forgot-password 忘记密码
Body: { "email": "string" }
POST /api/auth/reset-password 重置密码
Body: { "token": "string", "new_password": "string" }

📦 产品 Products

产品 CRUD、分类浏览、审核管理。支持分页浏览。

GET /api/products 产品列表
Query: category?, status?, search?, page=1, page_size=20
Auth: 可选（未登录可见已上架）
POST /api/products 创建产品 Bearer
角色: admin / member
Body: name, description, price, category, images?, tags?, stock?, brand?
GET /api/products/{id} 产品详情
Response: 产品完整信息（含图片、规格等）
PUT /api/products/{id} 更新产品 Bearer
角色: admin / member（仅限本人产品）
DELETE /api/products/{id} 删除产品 Bearer
功能: 软删除（is_deleted = true）

🔍 搜索 Search

多维产品搜索、语义向量搜索、搜索建议。支持中文分词和全文检索。

GET /api/search 产品搜索
Query: q?, category?, region?, min_price?, max_price?, sort_by=relevance, page=1, page_size=20, highlight=true
排序: relevance / price_asc / price_desc / newest
{
  "code": 200,
  "data": {
    "items": [...],
    "total": 42,
    "page": 1,
    "page_size": 20,
    "query": "农产品",
    "highlight": true
  }
}
GET /api/search/vector 语义向量搜索
Query: q (搜索文本), top_k=10
注意: 需启用 USE_VECTOR_SEARCH=1
POST /api/search/vector/rebuild 重建向量索引 Bearer
功能: 扫描全量数据重新生成向量索引

🤝 供需匹配 Matching

AI 驱动的供需智能匹配引擎，支持需求匹配产品和产品匹配需求。

GET /api/matching/needs/{id}/products 需求匹配产品
Query: strategy? (v1/v2), top_k=10
Response: 匹配结果列表含 match_score 和 match_reasons
GET /api/matching/products/{id}/needs 产品匹配需求
Query: strategy? (v1/v2), top_k=10
POST /api/matching/refresh 重建匹配索引 Bearer
功能: 刷新匹配引擎的倒排索引和 TF-IDF 模型

⭐ 推荐 Recommend

个性化推荐引擎，基于用户行为 + AI 匹配，提供热门推荐和个性化推荐。

GET /api/recommend/products 个性化产品推荐
Query: user_id?, limit=8
逻辑: 有行为→同类推荐，无行为→热门推荐
GET /api/recommend/products/{user_id} 按用户ID推荐
Query: limit=8
Response: personalized / hot 策略标识
GET /api/recommend/hot 热门推荐
Query: limit=10
算法: 过去7天浏览最多的产品
GET /api/recommend/personalized/{user_id} AI 个性化推荐（LLM）
功能: 结合 matching_engine + LLM 生成智能匹配理由
POST /api/recommend/feedback 推荐反馈
Body: { "user_id": int, "product_id": int, "action": "like"|"dislike", "source?": "string" }

📊 CRM 管道

商机管理、管道阶段追踪、活动记录。阶段：leads → qualified → proposal → negotiation → closed_won/closed_lost。

GET /api/crm/deals 商机列表 Bearer
Query: stage?, search?, owner_id?, page=1, page_size=20
POST /api/crm/deals 创建商机 Bearer
Body: title, value, stage, owner_id?, probability?, expected_close_date?, notes?
{
  "code": 201,
  "data": { "id": 1, "title": "...", "stage": "leads" }
}
GET /api/crm/deals/{deal_id} 商机详情 Bearer
Response: 包含商机完整信息和活动记录
PATCH /api/crm/deals/{deal_id} 更新商机 Bearer
Body: 部分更新 stage, value, probability, notes, expected_close_date 等
GET /api/crm/deals/{deal_id}/activities 商机活动记录 Bearer
Response: 该商机的所有活动时间线
GET /api/crm/pipeline 管道概览 Bearer
Response: 各阶段统计（商机数量、总金额）

🏢 企业数据 Enterprise

企业知识图谱 CRUD、搜索、关系图谱管理、公域数据采集补全。

GET /api/enterprise 企业列表 Bearer
Query: search?, industry?, region?, page=1, page_size=20
POST /api/enterprise 创建企业 Bearer

PUT /api/enterprise/{id} 更新企业 Bearer

📎 数据丰富 Enrich

企业信息丰富服务，通过企查查/天眼查等数据源自动补全企业信息（24小时缓存）。

GET /api/enrich/company 企业信息丰富 Bearer
Query: name (企业名称或关键词)
Response: 企业基本信息、经营范围、联系人信息
GET /api/enrich/company/basic 企业基本信息 Bearer
Query: name (企业名称)
Response: 工商基本信息（名称、信用代码、法人、注册资本等）
GET /api/enrich/company/scope 经营范围查询 Bearer
Query: name
Response: 企业经营范围、所属行业
GET /api/enrich/contacts 企业联系人 Bearer
Query: company (企业名称)
Response: 企业关键联系人信息

👥 多租户组织 Orgs

组织创建、成员管理、邀请管理。支持多团队协作。

POST /api/orgs 创建组织 Bearer
Body: { "name": "string" }
Response: 自动成为管理员
GET /api/orgs 我的组织列表 Bearer
GET /api/orgs/{org_id} 组织详情 Bearer
PUT /api/orgs/{org_id} 更新组织 Bearer
角色: 仅组织管理员
DELETE /api/orgs/{org_id} 删除组织 Bearer
角色: 仅组织拥有者
GET /api/orgs/{org_id}/members 成员列表 Bearer
POST /api/orgs/{org_id}/members 添加成员 Bearer
Body: { "user_id": int, "role": "member"|"admin" }
DELETE /api/orgs/{org_id}/members/{user_id} 移除成员 Bearer
PUT /api/orgs/{org_id}/members/{user_id}/role 变更成员角色 Bearer
Body: { "role": "admin"|"member" }
GET /api/orgs/{org_id}/invites 邀请列表 Bearer
POST /api/orgs/{org_id}/invites 创建邀请 Bearer
Body: { "email": "string" }
POST /api/orgs/invites/accept 接受邀请 Bearer
Body: { "token": "string" }

📈 增长引擎 Growth

邀请/推荐/分享机制，推广积分奖励系统。独立 SQLite 数据库。

POST /api/growth/invite 创建邀请链接 Bearer
Body: { "message": "string" }
Response: code, invite_url, expires_at
GET /api/growth/invites 邀请列表 Bearer
Query: page=1, page_size=20
GET /api/growth/invites/{code} 邀请详情
说明: 无需登录，供落地页使用
POST /api/growth/invites/accept 接受邀请 Bearer
Body: { "code": "string" }
奖励: 接受者 100 积分，邀请人 50 积分
GET /api/growth/stats 邀请统计 Bearer
Response: total_invited, total_accepted, total_reward
