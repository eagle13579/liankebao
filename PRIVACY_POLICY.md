# 链客宝AI 隐私政策

> 最后更新：2026-06-22 | 生效：2026-06-22

## 1. 引言
链客宝AI（"我们"）遵守《中华人民共和国个人信息保护法》（PIPL）、《数据安全法》、《网络安全法》及欧盟 GDPR 等法规，致力于保护您的个人信息安全。使用服务前请仔细阅读本隐私政策。

## 2. 我们收集的信息

### 2.1 您主动提供的信息
| 信息类型 | 具体内容 | 收集目的 |
|---------|---------|---------|
| 账户信息 | 手机号、微信昵称、头像 | 注册与身份认证 |
| 名片信息 | 姓名、公司、职位、联系方式 | 数字名片、人脉匹配 |
| 产品信息 | 名称、描述、价格、图片 | 商品展示与交易 |
| 订单信息 | 收货地址、订单详情 | 交易履约与物流 |
| 支付信息 | 支付凭证（不直接收集银行卡号） | 支付处理（微信支付/支付宝） |

### 2.2 自动收集的信息
- **设备信息**：型号、操作系统、唯一标识符（安全防护）
- **日志信息**：IP、访问时间、操作记录（故障排查）
- **位置信息**：IP级粗略位置（地域优化）

### 2.3 微信小程序 API
- `wx.login()`：临时登录凭证
- `wx.getUserProfile()`：昵称头像（需主动授权）
- `wx.getPhoneNumber()`：手机号（需主动授权）
- `wx.chooseAddress()`：收货地址（需主动授权）

## 3. 信息使用目的
1. **核心服务**：注册登录、名片生成、供需匹配、交易处理
2. **AI 服务**：智能匹配推荐、信任评分计算、需求预测
3. **安全保障**：身份验证、反欺诈检测、异常登录识别
4. **服务优化**：匿名化行为分析、功能改进
5. **法律合规**：满足监管要求

## 4. 数据存储与保护

### 4.1 存储
个人信息存储在中华人民共和国境内的阿里云服务器。

### 4.2 安全措施
- **加密传输**：HTTPS/TLS 1.3
- **加密存储**：AES-256-GCM
- **访问控制**：RBAC 最小权限原则
- **安全审计**：定期漏洞扫描和渗透测试
- **日志监控**：7×24 小时异常告警

### 4.3 保留期限
- 账户信息：注销后 30 天内删除
- 交易记录：依法保留至少 3 年
- 日志数据：保留 90 天

## 5. 信息共享
我们不出售您的个人信息。仅在以下情况共享：
- **微信支付/支付宝**：订单金额（支付处理）
- **物流服务商**：收货地址（商品配送）
- **云服务商**：加密后的数据（服务器托管）
- **监管机构**：依法要求的信息

## 6. 您的权利（PIPL & GDPR）
| 权利 | 行使方式 |
|------|---------|
| 访问权 | App内「我的→隐私中心→数据下载」 |
| 更正权 | App内「我的→编辑资料」 |
| 删除权 | App内「我的→设置→注销账户」 |
| 数据可携权 | App内「我的→隐私中心→数据导出」 |
| 撤回同意 | App内「我的→隐私中心→授权管理」 |

## 7. 未成年人保护
我们不对 14 周岁以下未成年人提供服务。如无意收集了未成年人信息，将立即删除。
