## 启动方式

```bash
# 后端
cd /mnt/d/链客宝/backend
pip install -r requirements.txt
python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

# 前端
cd /mnt/d/链客宝/frontend
npm install
npm run dev
```

**访问**：后端 `http://localhost:8000` | 前端 `http://localhost:5173` | API文档 `http://localhost:8000/docs`
